# Customer Churn Prediction

## Models Used: 
The following models are implemented in the notebook:
- Logistic Regression
- LightGBM
- XGBoost
- Decision Tree Classifier
- Random Forest Classifier
- Naive Bayes
- KNN

## Evaluation Metric
Model performance is evaluated using:
- Accuracy
- Confusion Matrix

## Dataset
The dataset consists of flight-related information such as:
- ID
- Customer ID
- Last Name
- Credit Score
- Country
- Gender
- Age
- Tenure
- Account Balance
- Production Count
- Has Card
- Is Active
- Estimated Salary
- Exit Status (target variable)
